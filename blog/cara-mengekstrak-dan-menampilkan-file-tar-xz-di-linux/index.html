✪ Menampilkan apa saja isi didalam file .tar.xz

Buka file manager sampai berada di lokasi file .tar.xz tersebut berada, kemudian klik kanan dan klik Open Terminal. Kemudian ketik script berikut ini di terminal lalu tekan enter.

tar -tf namafile.tar.xz

Terminal akan menampilkan apa saja file yang ada didalam file .tar.xz tersebut.

✪ Mengekstrak file .tar.xz

Ketik script berikut ini di terminal lalu tekan enter.

tar -xf namafile.tar.xz

Akan muncul folder baru yang berisi file-file hasil ekstrak dari file .tar.xz tersebut.

Apabila ada error atau gagal melakukan ekstrak, biasanya karena tar dan xz belum terpasang di komputermu. Silahkan install keduanya terlebih dahulu.
